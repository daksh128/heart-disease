# heart-disease

❤️ Heart Disease Prediction System

📌 About

This project is a Machine Learning model that predicts whether a person has heart disease based on medical data.

🛠️ Tools Used :-
Python
Pandas
NumPy
Scikit-learn

▶️ How to Run
Install libraries:
pip install numpy pandas scikit-learn

Run the program:
python heart.py
Enter the required details to get prediction.

📊 Features
Predicts heart disease using Logistic Regression
Takes user input from console
Shows training and testing accuracy

📂 Files
heart.py → main code
heart_disease_data.csv → dataset

💡 Example
Enter Age: 45  
Enter Sex: 1  
Enter Chest Pain Type: 2  
...

Prediction Result:
No Heart Disease
